- Create tasks
- Move tasks
- Copy tasks
- Delete tasks

## Run Locally

Clone the project

```bash
  git clone https://github.com/alejandrofloresmtz94/zustand_example.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev   
```
